# KB Save

Save content to the Open Notebook knowledge base.

## Usage

```bash
kb-save "<url_or_text>"
```

## Examples

- "Save this article: https://example.com/article" → `kb-save "https://example.com/article"`
- "Save a note: meeting moved to Thursday" → `kb-save "meeting moved to Thursday"`

Confirm after saving: "Saved to knowledge base."
