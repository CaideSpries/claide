# Birthday Add

Add a person's birthday to the tracker.

## Usage

1. Extract: name and date (month + day, year optional)
2. Append to `workspace/data/birthdays.txt`
3. Format: `YYYY-MM-DD | Name` (use 0000 for year if unknown)
4. Confirm: "Added [name]'s birthday on [date]"
