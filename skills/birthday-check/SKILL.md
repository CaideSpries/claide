# Birthday Check

Read `workspace/data/birthdays.txt` and report any birthdays in the next 7 days.

## Output Format

If birthdays found:
```
Upcoming birthdays:
- [Name] — [Date] ([days until])
```

If none: "No upcoming birthdays in the next 7 days."
