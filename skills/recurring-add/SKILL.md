# Recurring Add

Add a recurring reminder to the tracker.

## Usage

1. Extract: what, frequency (daily/weekly/monthly), time, channel
2. Append to `workspace/data/recurring.txt`
3. Format: `frequency | HH:MM | channel | message`
4. Confirm: "Recurring reminder set: [what] [frequency] at [time]"
